## 常用的逻辑等价(1)

| 名称       | 等价 |
|------------|------|
| 双重否定律 | p ≡ ¬¬p |
| 幂等律     | p ≡ p ∨ p, p ≡ p ∧ p |
| 交换律     | p ∨ q ≡ q ∨ p, p ∧ q ≡ q ∧ p |
| 结合律     | (p ∨ q) ∨ r ≡ p ∨ (q ∨ r), (p ∧ q) ∧ r ≡ p ∧ (q ∧ r) |
| 分配律     | p ∨ (q ∧ r) ≡ (p ∨ q) ∧ (p ∨ r), p ∧ (q ∨ r) ≡ (p ∧ q) ∨ (p ∧ r) |
| 德摩根律   | ¬(p ∨ q) ≡ ¬p ∧ ¬q, ¬(p ∧ q) ≡ ¬p ∨ ¬q |
| 吸收律     | p ∨ (p ∧ q) ≡ p, p ∧ (p ∨ q) ≡ p |

## 常用的逻辑等价(2)

| 名称       | 等价 |
|------------|------|
| 支配律     | A ∨ 1 ≡ 1, A ∧ 0 ≡ 0 |
| 恒等律     | A ∨ 0 ≡ A, A ∧ 1 ≡ A |
| 排中律     | A ∨ ¬A ≡ 1 |
| 矛盾律     | A ∧ ¬A ≡ 0 |
| 假言易位   | A → B ≡ ¬B → ¬A |
| 归谬论     | (A → B) ∧ (A → ¬B) ≡ ¬A |

## 逻辑等价的判定
- ¬(p → q) 和 p ∧ ¬q 是否逻辑等价？
  - ¬(p → q) ≡ ¬(¬p ∨ q) ≡ ¬(¬p) ∧ ¬q ≡ p ∧ ¬q
- p ∧ q → p ∨ q 是否永真？
  - p ∧ q → p ∨ q ≡ ¬(p ∧ q) ∨ (p ∨ q) ≡ (¬p ∨ ¬q) ∨ (p ∨ q) ≡ ¬p ∨ p ∨ ¬q ∨ q ≡ T

## 通过逻辑等价进行推理
已知：
- ((p1 ∧ ¬p2) ∨ (¬p1 ∧ p2)) ∧ ((p3 ∧ ¬p4) ∨ (¬p3 ∧ p4)) ∧ ((p5 ∧ ¬p6) ∨ (¬p5 ∧ p6)) 为真
- 通过逻辑等价推导得出：¬p1 ∧ p2 ∧ p3 ∧ ¬p4 ∧ ¬p5 ∧ p6 为真
- 结论：Jim is from Chicago, Sam is from Boston, and Bill is from Detroit.

## 命题逻辑公式的范式
- 为何要“范式”？
  - 真值表方法在命题变元数目较大时计算量大
  - 同一问题可以有不同但等价的命题公式表达形式
  - 需要使命题公式规范化

### 术语
- 文字：命题变元或命题变元的否定
- 简单析取式（基本和）：有限个文字的析取式
- 简单合取式（基本积）：有限个文字的合取式
- 析取范式（DNF）：由有限个简单合取式构成的析取式
- 合取范式（CNF）：由有限个简单析取式构成的合取式

### 性质
- 一个文字既是析取范式又是合取范式
- 析取范式为矛盾式当且仅当每个简单合取式是矛盾式
- 合取范式为重言式当且仅当每个简单析取式是重言式

### 定理
- 任一命题公式都存在着与之等值的析取范式和合取范式
- 构造方法：
  1. 使公式中仅含联结词¬, ∧, ∨
  2. 将否定符¬移到命题变元前
  3. 利用分配律将公式化成析取范式或合取范式

### 主析取范式和主合取范式
- 极小项：包含所有命题变元或其否定一次仅一次的简单合取式
- 极大项：包含所有命题变元或其否定一次仅一次的简单析取式
- 主析取范式：由有限个极小项组成的析取范式
- 主合取范式：由有限个极大项组成的合取范式

### 极小项和极大项的性质
- 没有两个不同的极小项（极大项）是等价的
- 每个极小项只有一组真值指派使其为真
- 每个极大项只有一组真值指派使其为假
- 任意两极小项的合取必假，任意两个极大项的析取必为真
- 极大项的否定是极小项，极小项的否定是极大项
- 所有极小项的析取为永真公式，所有极大项的合取是永假公式

### 主析取范式和主合取范式的存在性和唯一性
- 任何命题公式的主析取范式和主合取范式存在且唯一

### 利用真值表技术求主析取范式和主合取范式
1. 选出公式的真值结果为真的所有行，找到对应的极小项，析取得到主析取范式
2. 选出公式的真值结果为假的所有行，找到对应的极大项，合取得到主合取范式

## 命题逻辑的推理问题

### 论证
- 论证：命题序列，除最后一个命题外称为前提，最后一个命题称为结论
- 论证有效：所有前提为真时结论必为真

### 论证形式
- 论证形式：涉及命题变元的复合命题序列
- 论证形式有效：无论用什么命题替换命题变元，前提为真时结论必为真
