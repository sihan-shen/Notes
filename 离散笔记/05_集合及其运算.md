## 集合及其运算

### 基本概念
#### 集合的定义
- 直观定义：一个集合是一组无序的对象，这些对象称为这个集合的元素或成员
- Georg Cantor的描述：A set is a collection into a whole of definite, distinct objects of our intuition or our thought

#### 集合的描述
1. 外延法：罗列、枚举
   - 例：V={a, e, i, o, u}
2. 概括法：{x | P(x)}
   - 例：ℤ⁺={x∈ℤ | x>0}
3. 文氏图（Venn diagrams）

#### 集合相等与子集关系
- 集合相等：A=B ⇔ ∀x(x∈A ↔x∈B)
- 子集：A⊆B ⇔ ∀x(x∈A → x∈B)
- 真子集：A⊂B ⇔ A⊆B ∧ A≠B
- 定理：A=B ⇔ A⊆B ∧ B⊆A

#### 集合的大小
- 有限集合：恰有n个不同元素，基数|S|=n
- 无限集合：不是有限的集合

#### 空集
- 定义：没有任何元素的集合，记作∅
- 性质：
  - 空集是任何集合的子集
  - 空集是唯一的

#### 幂集
- 定义：S的幂集是S的所有子集的集合，记作℘(S)或2^S
- 性质：若|A|=n，则|℘(A)|=2^n

### 集合运算
#### 基本运算
1. 并：A∪B={x | x∈A ∨ x∈B}
2. 交：A∩B={x | x∈A ∧ x∈B}
3. 相对补（差）：A-B={x | x∈A ∧ x∉B}
4. 补集（全集U下）：~B=U-B
5. 对称差：A⊕B=(A-B)∪(B-A)

#### 广义运算
1. 广义并：⋃𝓐={x | ∃A(A∈𝓐 ∧ x∈A)}
2. 广义交（𝓐非空）：⋂𝓐={x | ∀A(A∈𝓐 → x∈A)}

#### 运算性质
- 最小上界（并）：
  - A⊆A∪B, B⊆A∪B
  - 若A⊆X, B⊆X，则A∪B⊆X
- 最大下界（交）：
  - A∩B⊆A, A∩B⊆B
  - 若X⊆A, X⊆B，则X⊆A∩B

### 集合恒等式
| 等式 | 名称 |
|------|------|
| A∪∅=A, A∩U=A | 恒等律 |
| A∪U=U, A∩∅=∅ | 支配律 |
| A∪A=A, A∩A=A | 幂等律 |
| ~(~A)=A | 补集律 |
| A∪B=B∪A, A∩B=B∩A | 交换律 |
| A∪(B∪C)=(A∪B)∪C, A∩(B∩C)=(A∩B)∩C | 结合律 |
| A∩(B∪C)=(A∩B)∪(A∩C), A∪(B∩C)=(A∪B)∩(A∪C) | 分配律 |
| ~(A∪B)=~A∩~B, ~(A∩B)=~A∪~B | 德摩根定律 |
| A∪(A∩B)=A, A∩(A∪B)=A | 吸收律 |
| A∪~A=U, A∩~A=∅ | 补律 |

### 自然数的构造
#### 皮亚诺公理
1. 零是个自然数
2. 每个自然数都有一个后继
3. 零不是任何自然数的后继
4. 不同的自然数有不同的后继
5. 归纳公理

#### 集合定义
- 后继：s(a)=a∪{a}
- 自然数集合ℕ：所有归纳集的交集
  - 0=∅
  - 1={∅}
  - 2={∅,{∅}}
  - 3={∅,{∅},{∅,{∅}}}

#### 运算定义
- 加法：
  - m+0=m
  - m+s(n)=s(m+n)
- 乘法：
  - m*0=0
  - m*s(n)=m+m*n
- 序关系：a≤b ⇔ ∃c∈ℕ. a+c=b

### 集合相关命题的证明方式
1. 直接使用集合包含、相等定义
2. 利用运算定义作逻辑等值式推演
3. 利用已知恒等式或等式作集合代数推演
4. 利用成员表证明集合恒等式

### 笛卡尔乘积
- 有序偶：(a,b)=(x,y) ⇔ a=x ∧ b=y
- 笛卡尔积：A×B={(a,b)| a∈A ∧ b∈B}
- n元笛卡尔积：A₁×A₂×...×Aₙ={(a₁,a₂,...,aₙ)| aᵢ∈Aᵢ}

### 集合悖论
- Russell悖论：定义R={x | x∉x}，则R∈R ⇔ R∉R
- 理发师悖论："我给所有不给自己理发的人理发"
